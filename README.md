# City-Mobility-NYC Dashboard
## 📊 Project Overview
This project analyzes **1.45 Million+ NYC taxi records** to optimize operational efficiency. Using Power BI, I transformed massive raw data into strategic business insights.

## 🛠️ Technical Highlights
- **Big Data Cleaning:** Processed 1.4M rows in Power Query; handled outliers and GPS noise via Geofencing.
- **Precision Metrics:** Validated a **13.99-minute average trip duration** as the operational benchmark.
- **Advanced Visuals:** Implementation of **Report Page Tooltips** and Heatmaps for granular demand analysis.

## 💡 Key Insights
- **Peak Demand:** Friday at 6:00 PM is the highest revenue window.
- **Geo-Focus:** 70% of trips originate in Midtown Manhattan.
- **Efficiency:** Standardized ETAs at 14 mins to reduce customer cancellations.
